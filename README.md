# RoundedAndShadowButton
Swift UIButton setting Rounded and Shadow using IBInspectable

it is **make easy** UIButton rounded or shadow

@IBInspectable list
* borderColor (UIColor)
* borderWidth (CGFloat)
* cornerRadius (CGFloat)

* shadowOpacity (Float)
* shadowRadius (CGFloat)
* shadowColor (UIColor)
